Create Symlink

Requried FollowSymlinks option: ln -s /path/to/zf/public /var/www/htdocs

Check directories

Run index.sh script Directories cache, logs and captcha (for forms) should be writable: chmod a+w /path/to/zf/data/*
chmod a+w /path/to/zf/public/captcha/

Set environment

Open file public/.htaccess and change APPLICATION_ENVIRONMENT define to site environment,
commonly one of "development", "staging", "testing", "production", but wholly dependent on your organization's and/or site's needs.

Configuration file is requried (see application/config/application.yaml).

Run DB Migrations

Open directory bin and run migration: cd bin/ ./zfc.sh up migration

Run Unit Tests

Unit tests aviable on CLI cd /path/to/zf/tests/
# Run main test suite
phpunit ./
 For generate reports you should be remove comments from "logging" part of phpunit.xml and create directory tests/reports with permissions for write
Reports is available on follow link: tests/reports
